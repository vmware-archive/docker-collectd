# docker-collectd

Builds a docker image for running collectd in container.

## Requirements

A working Docker setup and a valid shell (e.g., bash).

## Getting Started
Build the container and run it.

## Example

```
./build.sh
docker run -d vmware-opencloud/collectd.
```
